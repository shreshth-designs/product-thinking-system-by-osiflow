# Agents — How to use these
# Open Orbit by Osiflow

These are standalone AI agents for product thinking.
Each one is a different expert in the room.

---

## How to use any agent

### In Claude Code:
Open this project folder. The agents auto-load with the CLAUDE.md context.
Just describe your problem — Claude will respond as the right agent.

### In Claude chat (no Claude Code needed):
1. Open the agent file you want
2. Copy everything in it
3. Paste into Claude chat
4. Then write your question or paste your brief

---

## Which agent for what

| You want to... | Use this agent |
|----------------|---------------|
| Get a PM's honest take on your decision | pm-reviewer.md |
| Hear how a real user would respond | user-researcher.md |
| Find every way your idea could fail | devils-advocate.md |
| Think about the bigger strategic picture | product-strategist.md |

---

## Recommended sequence for a new problem

1. Start with /problem-framer (command) — frame the problem
2. Run pm-reviewer — get the PM perspective
3. Run user-researcher — hear the user perspective
4. Run devils-advocate — find the failure modes
5. Run /adversarial-review (command) — final check before handing off
