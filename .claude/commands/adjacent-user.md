# Adjacent User Finder
# Open Orbit by Osiflow — Product Thinking System

Ask me to describe my product in 2 sentences. Wait for it.

Step 1: "Who tried this and stopped?"
List 3 types. For each: most likely friction point.

Step 2: "Who almost signed up but didn't?"
What one thing probably stopped them?

Step 3: "Which of these is easiest to convert with a single change?"
Name the change specifically — not "improve onboarding", say what changes on which screen.

Step 4: "What would I design differently if I focused on this group?"
Give 2 concrete differences.

This is not about personas. It's about the gap between who uses the product and who almost uses it.
That gap is where growth lives.
